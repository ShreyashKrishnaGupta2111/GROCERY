# GROCERY
GROCERY

A high-energy grocery ordering landing page inspired by quick-commerce experiences such as Blinkit. It includes a sticky navigation bar, public-attention offer ticker, animated hero section, grocery categories, lightning deal cards, and an interactive cart drawer.

## Run locally

Open `index.html` in a browser, or serve the folder with any static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
